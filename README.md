# E-Commerce Data Analysis Dashboard

## Deskripsi
Proyek ini bertujuan untuk menganalisis data e-commerce guna mengetahui tren penjualan dan kategori produk terlaris berdasarkan periode waktu tertentu.

## Dataset
Dataset yang digunakan merupakan data transaksi e-commerce yang mencakup informasi produk, waktu pembelian, dan harga.

## Pertanyaan Bisnis
1. Kategori produk apa yang memiliki jumlah penjualan terbanyak selama periode tahun 2017 hingga 2018?
2. Bagaimana tren total revenue bulanan selama periode tahun 2017 hingga 2018?

## Insight
- Kategori produk dengan jumlah penjualan tertinggi adalah **bed_bath_table**, yang menunjukkan bahwa kategori ini paling diminati oleh pelanggan.
- Revenue bulanan menunjukkan fluktuasi yang signifikan, dengan puncak tertinggi terjadi pada bulan November 2017 dan terendah pada bulan Desember 2016.
- Peningkatan revenue pada akhir tahun mengindikasikan adanya pengaruh faktor musiman atau promosi terhadap perilaku pembelian pelanggan.

## Cara Menjalankan Dashboard
1. Install dependencies:
pip install -r requirements.txt

2. Jalankan Streamlit:
streamlit run dashboard/dashboard.py
