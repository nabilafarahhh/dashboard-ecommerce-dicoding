# E-Commerce Data Analysis Dashboard

## Deskripsi
Proyek ini bertujuan untuk menganalisis data e-commerce guna mengetahui tren penjualan dan kategori produk terlaris.

## Dataset
Dataset yang digunakan merupakan data transaksi e-commerce yang mencakup informasi produk, waktu pembelian, dan harga.

## Pertanyaan Bisnis
1. Kategori produk apa yang paling banyak terjual?
2. Bagaimana tren revenue dari waktu ke waktu?

## Insight
- Kategori produk tertentu memiliki jumlah penjualan paling tinggi.
- Revenue mengalami fluktuasi berdasarkan waktu.

## Cara Menjalankan Dashboard
1. Install dependencies:
pip install -r requirements.txt

2. Jalankan streamlit:
streamlit run dashboard/dashboard.py
